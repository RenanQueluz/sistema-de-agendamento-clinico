# Sistema-de-agendamento-clinico

Sistema web Full Stack desenvolvido para gerenciamento de consultas médicas, permitindo o cadastro de pacientes e o controle organizado de agendamentos em uma clínica.

---

Sobre o projeto

O **Sistema de Agendamento Clínico** foi desenvolvido com o objetivo de simular um sistema real utilizado em clínicas médicas, aplicando conceitos práticos de desenvolvimento web, criação de APIs REST e integração com banco de dados.

O projeto busca resolver o problema da organização manual de consultas, oferecendo uma interface simples para cadastro e gerenciamento de atendimentos.

---

Funcionalidades

Cadastro de pacientes
Agendamento de consultas
Seleção de plano de saúde Listagem de atendimentos
Organização por data e horário
Integração com banco de dados
Comunicação Front-end ↔ Back-end via API

---

Tecnologias utilizadas

Front-end

* HTML5
* CSS3
* JavaScript

Back-end

* Node.js
* Express.js

Banco de Dados

* MySQL

---

Conceitos aplicados

* Criação de API REST
* Métodos HTTP (GET, POST, PUT, DELETE)
* Manipulação de JSON
* Integração com banco de dados relacional
* Organização MVC básica
* Comunicação cliente-servidor

---

Objetivo do projeto

Este projeto foi desenvolvido como parte do aprendizado em **Análise e Desenvolvimento de Sistemas**, com foco de aprender na pratica a trabalhar na construção de aplicações web completas e preparação para atuação profissional como desenvolvedor Full Stack.
