# Sistema-de-agendamento-clinico

Sistema web Full Stack desenvolvido para gerenciamento de consultas médicas, permitindo o cadastro de pacientes e o controle organizado de agendamentos em uma clínica.

---

Sobre o projeto

O **Sistema de Agendamento Clínico** foi desenvolvido com o objetivo de simular um sistema real utilizado em clínicas médicas, aplicando conceitos práticos de desenvolvimento web, criação de APIs REST e integração com banco de dados.

O projeto busca resolver o problema da organização manual de consultas, oferecendo uma interface simples para cadastro e gerenciamento de atendimentos.

---

Funcionalidades

Cadastro de pacientes
Agendamento de consultas
Seleção de plano de saúde Listagem de atendimentos
Organização por data e horário
Integração com banco de dados
Comunicação Front-end ↔ Back-end via API

---

como deve funcionar na pratica

1. O usuario entra no sistema e entra com nome e senha caso tenha cadastro se não deve se cadastrar.

2. Em cadastro o usuario deve preencher os campos com as informações requisitadas e pronto esta cadastrado no sistema.

3. Logo após a entrada do usuario, ele sera direcionado a uma pagina aonde todos os agendamentos do dia estarão visiveis para exemplo: se hoje é dia 08/04/2026 automaticamente os agendamentos dessa data sera exibida sem precisar consultar uma busca

4.Caso o usuario queira cria um novo agendamento ele pode clicar no botão novo agendamento que sera direcionado para preencher um formulario com as informções nescessarias para o agendamento do paciente

5. Se o usuario desejar fazer buscas especificas por pacientes logo abaixo do formulario, ele pode pesquisar a data que foi marcada na consulta e encontrar o paciente expecifico (em projetos futuros colocarei opções de pesquisar por nome para uma busca mais elaborada)

6. ao clicar em pesquisar sera exiba uma tabela como todos os agendamentos com data que digitada, com a opção deletar ou editar as informações sobre o paciente.
